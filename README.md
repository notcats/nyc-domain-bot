# 🏙️ NYC Domain Bot

Telegram бот для мониторинга истекающих доменов с NYC-тематикой.

## Возможности

- `/add <домен>` — добавить домен в список слежения
- `/list` — показать все домены с датами истечения
- `/check <домен>` — разовая WHOIS-проверка
- `/remove <домен>` — убрать домен из списка
- Ежедневный алерт в 09:00 МСК, если до истечения ≤ 30 дней

## Индикаторы

| Цвет | Значение |
|------|----------|
| 🟢 | > 30 дней |
| 🟡 | ≤ 30 дней |
| 🔴 | ≤ 7 дней или уже истёк |

## Деплой на Railway

1. **Fork/import** этого репо в Railway
2. Добавьте **переменную окружения** `BOT_TOKEN` в настройках сервиса
3. (Опционально) Добавьте **Volume** и укажите `DATA_PATH=/data` — тогда список доменов сохраняется между деплоями
4. Railway автоматически запустит `node bot.js`

> **Важно:** Без Volume файл `data.json` хранится внутри контейнера и сбрасывается при каждом деплое.

## Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `DATA_PATH` | Путь к Railway Volume (напр., `/data`). Если не указан, данные хранятся рядом с кодом |

## Локальный запуск

```bash
npm install
cp .env.example .env
# Вставьте BOT_TOKEN от @BotFather в .env
node bot.js
```
