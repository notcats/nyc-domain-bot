# 🏙️ NYC Domain Bot

Telegram бот для мониторинга истекающих доменов с NYC-тематикой.

## Возможности

- `/add <домен>` — добавить домен в список слежения
- `/list` — показать все домены с датами истечения
- `/check <домен>` — разовая WHOIS-проверка
- `/remove <домен>` — убрать домен из списка
- Ежедневный алерт в 09:00 МСК, если до истечения ≤ 30 дней

## Индикаторы

| Цвет | Значение |
|------|----------|
| 🟢 | > 30 дней |
| 🟡 | ≤ 30 дней |
| 🔴 | ≤ 7 дней или уже истёк |

## Запуск

```bash
npm install
cp .env.example .env
# Вставьте BOT_TOKEN от @BotFather в .env
node bot.js
```

## Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `BOT_TOKEN` | Токен бота от @BotFather |
